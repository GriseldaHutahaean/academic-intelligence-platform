# Academic Intelligence Platform

Academic Intelligence Platform is a data analytics project focused on understanding student academic outcomes and identifying factors associated with student success, retention, and dropout risk.

The project aims to transform educational data into actionable insights through data analysis, visualization, predictive modeling, and dashboard development.

## Objectives

* Explore patterns in student academic performance.
* Identify factors related to student success and dropout.
* Perform exploratory data analysis (EDA).
* Develop predictive models for academic outcome prediction.
* Present insights through interactive dashboards.

## Dataset

This project uses the **Students Dropout and Academic Success** dataset from Kaggle.

The dataset contains information related to student demographics, academic background, enrollment characteristics, and academic outcomes.

Target outcomes include:

* Graduate
* Enrolled
* Dropout
